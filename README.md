Satellite Application
### Сборка

Требуется Java 17 + postgres + доступ к браузеру

### Перед началом использования приложения необходимо переопределить следующие переменные в файле application.properties:
| Переменная                         | Описание                                               | Пример                                                    |
|------------------------------------|--------------------------------------------------------|-----------------------------------------------------------|
| server.port                        | порт, на котором запускается приложение                | 8092                                                      |
| spring.datasource.url              | путь подключения к базе данных                         | jdbc:postgresql://localhost:5432/user?currentSchema=links |
| spring.datasource.username         | имя пользователя базы данных                           | test                                                      |
| spring.datasource.password         | пароль для подключения к базе данных                   | test                                                      |
| spring.flyway.schemas              | имя схемы для миграции данных                          | links                                                     |
| short-links.properties.amountToAdd | Количественное представление времени действия ссылки.  | 1                                                         |
| short-links.properties.unit        | Мера измерения для вычисления времени действия ссылки. | DAYS                                                      |
| short-links.properties.remainder   | Количество переходов по ссылке.                        | 10                                                        |


### Описание пользовательского поведения
##### Генерация ссылки
1. Открыть страницу /generate
2. Введите имя пользователя и оригинальную ссылку. Если не вносить данные в оставшиеся поля, то они заполнятся автоматически
3. Нажмите на кнопку **Сгенерировать** и сохраните полученные uuid пользователя и короткую ссылку

##### Получение ссылки
1. Открыть страницу /get-link
2. Введите uuid + короткую ссылку в соответствующие поля и нажмите на кнопку **Получить ссылку**

##### Редактирование ссылки
1. Открыть страницу /edit-link
2. Введите uuid + короткую ссылку в соответствующие поля и нажмите на кнопку **Заполнить**
3. Оставшиеся поля заполнятся теми значениями, что хранятся в БД
4. Введите изменения и нажмите на кнопку **Редактировать**


### Алгоритм вычисления короткой ссылки
1. Складывается оригинальная ссылка, имя пользователя и время системы
2. Вычисляется хэш полученной строки
3. Из полученного хэша берутся первые 8 символов
